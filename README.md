# jest-puppeteer
> Jest + Puppeteer Scaffolding project for UI and API testing

## Test all

```bash
npm i && npm test
```

## Test only api

```bash
npm test -- api
```
## Test only web

```bash
npm test -- wev
```

## Add more config

To add more tests please add it in config/jest and modify jest.config.js
